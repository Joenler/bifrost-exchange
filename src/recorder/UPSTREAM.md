## Program.cs (replaces Phase 00 skeleton)

Rewired from `ArenaRecorder/Program.cs`:

- Switched from `WebApplication.CreateBuilder` to `Host.CreateApplicationBuilder` — no HTTP surface on the recorder for Phase 02.
- Dropped Arena's OpenTelemetry pipeline + OTLP exporter.
- Dropped `Analysis/` + `Api/` references + every `app.MapGet` for REST endpoints.
- Added `SchemaMigrator.ApplyPending()` call BEFORE `AddHostedService<WriteLoop>()` (Pitfall 7: migrations must complete on the foreground startup thread; the hosted write loop starts draining afterwards).
- Added `SqlMapper.AddTypeHandler(new DecimalTypeHandler())` + `AddTypeHandler(new BoolTypeHandler())` at startup (Pitfall 6: type handlers are process-global and must register before any Dapper call executes).
- Added a `IHostApplicationLifetime.ApplicationStopping` callback running `PRAGMA wal_checkpoint(TRUNCATE)` + stamping `Manifest.EndTime` / `Manifest.ExitReason` via `ExitReasonDetector` (T-02-09 mitigation for data-loss-on-shutdown).
- Preserved the Phase 00 `StartupLogger` sentinel (writes `/tmp/bifrost-ready` for the Docker healthcheck); runs alongside the new hosted services.
